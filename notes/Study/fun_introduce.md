
# 美食视频点播系统 - 主要功能说明

## 1. 用户与账号模块

- **游客访问**: 未登录用户可浏览公开视频列表、查看视频详情等基础信息,但不能点赞。  
- **账号注册与登录**: 支持邮箱/账号密码登录，登录后生成会话并保持登录状态。  
- **用户信息管理**: 登录用户可修改昵称、头像、密码等个人资料。    
- **关注关系**: 支持关注 / 取消关注 UP 主，维护粉丝与关注列表。

## 2. 视频内容管理

- **视频上传**: 登录用户可以上传本地视频文件并填写标题、简介、分类、标签等信息。  
- **视频审核**: 管理员对新上传视频进行审核，通过后上架，未通过则驳回。  
- **视频上下架**: 管理员可对已审核视频进行上架 / 下架管理。  
- **视频删除**: 视频作者或管理员可删除视频，同时清理相关业务数据。  
- **视频分类与标签**: 每个视频可关联分类和标签，便于后续分类浏览与筛选。

## 3. 视频播放与互动

- **视频播放**: 用户可以在线播放已上架的视频，系统记录播放次数。  
- **点赞功能**: 登录用户可对视频进行点赞 / 取消点赞，并统计点赞数量。  
- **弹幕功能**: 登录用户在播放时可发送弹幕，其他用户可实时查看同一视频的弹幕列表。  
- **播放数据统计**: 记录点赞量等数据，用于展示视频热度。

## 4. 视频列表与推荐

- **首页视频列表**: 展示最新或热门视频列表，支持分页浏览。  
- **个人视频列表**: 支持按用户查看其发布的视频列表。  
- **分类视频列表**: 支持按视频分类查看对应的视频列表。  
- **标签视频列表**: 支持按标签查看对应的视频列表。  
- **状态视频列表**: 管理端可按审核状态、上架状态等维度查看视频列表。

## 5. 搜索与检索

- **关键词搜索**: 用户可通过关键字搜索视频标题 / 简介相关的视频。  
- **搜索结果过滤**: 仅展示已上架且通过审核的视频，并支持分页查看搜索结果。

## 6. 后台管理功能

- **用户管理**: 管理员可查看用户列表、查询指定用户信息，并对其进行禁用 / 启用等操作。  
- **管理员管理**: 支持新增、编辑、删除管理员账号，控制不同管理员的权限等级。  
- **视频审核管理**: 集中查看待审核视频、审核通过或驳回，并查看审核历史。  
- **运营统计概览（可选）**: 汇总用户数、视频数、播放量、点赞量等指标，为运营分析提供数据参考。
